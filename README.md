* 정렬
  1. 선택 정렬
    - 가장 기본적인 정렬 
    - 가장 작은 값을 찾아 맨뒤나 앞에 두는 정렬 
    - 반복을 하나씩 줄여가면서 정렬하는 방식 
    - 시간복잡도는 O(N제곱);
    - 정확히 말하면 N*(N+1)/2
  
  2. 버블 정렬
    - 구현하기에 가장 쉬운 정렬 
    - 하지만 다른 정렬에 비해 가장 비효율적
    - 시간복잡도는 선택정렬과 동일하게 O(N*N)
